# Приложение для прогнозирования акций энергетического сектора 📈🛢️

Веб-приложение на базе машинного обучения для прогнозирования ценовых тенденций акций в нефтяном секторе. Приложение, созданное с использованием Python и Streamlit, предоставляет интерактивный интерфейс для анализа данных и использует The Guardian API для получения актуальных рыночных новостей.

Проект полностью контейнеризирован с помощью Docker, что обеспечивает простоту развертывания и запуска в любой среде без необходимости ручной настройки зависимостей.

## Ключевые возможности
* **Интерактивный интерфейс:** Удобный и адаптивный дашборд, созданный на Streamlit.
* **Интеграция машинного обучения:** Использование исторических рыночных данных для прогнозирования трендов в энергетическом секторе.
* **Новостной контекст:** Интеграция с The Guardian API для отображения релевантных статей и понимания контекста движений рынка.
* **Контейнеризация:** Простая и воспроизводимая установка с помощью Docker.

## Технологический стек
* **Язык программирования:** Python 3.11
* **Фреймворк:** Streamlit
* **Обработка данных и ML:** Pandas, Scikit-Learn
* **Инфраструктура:** Docker

## Требования
Перед запуском проекта убедитесь, что у вас есть:
* Установленный [Docker](https://docs.docker.com/get-docker/).
* Бесплатный API-ключ от [The Guardian Open Platform](https://open-platform.theguardian.com/).

## Установка и запуск

**1. Клонируйте репозиторий:**
```bash
git clone https://github.com/dalishev/stock_predictor.git
cd energy-stock-predictor
```

**2. Соберите Docker-образ:**

**Стандартные ОС:**
```bash
docker build -t stock_predictor .
```
**Линукс:**
```bash
docker build --platform=linux/arm64 -t stock_predictor .
```

**3. Запустите контейнер:**
Запустите приложение, передав ваш API-ключ Guardian в качестве переменной в данной команде:

```bash
docker run -p 8501:8501 -e GUARDIAN_API_KEY="ваш_ключ_сюда" stock_predictor
```

**4. Откройте браузер и перейдите по этой ссылке**
```bash
localhost:8501
```
